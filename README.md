# elliptic-curve

Elliptic curve library for Haskell.

## About

Haskell の型レベルの機能をいい感じに使った楕円曲線上の有理点演算を提供したい．

## Feature

- 標数の情報を型レベルで持つ有限体型を実装．
  - 今のところ素体のみそれっぽいものができている．
  - 型レベルでの素数判定等も考えているが，あまりに遅いので要検討．

- 拡大体型の実装．
  - 多項式演算や Conway 多項式生成の実装が必要．

- 型レベル有理数を実装．

- 楕円曲線の情報を型レベルで実現．
  - 楕円曲線の標準形における係数の情報を型に持たせる．
